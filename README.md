# Classic Faithful 64x with full modded support

Just as the title suggests, this project is a collection of 64x modded Minecraft textures, made by myself, based on the Classic Faithful 64x resource pack. 

It is recommended to use this pack along with the Faithful 32x Modded resource pack for the best experience. When installing, make sure Faithful 64x Modded is listed above Faithful 32x modded in your enabled resource pack list.

This pack includes textures for vanilla Minecraft. They do not need to be installed separately, however, it is optional if you would like to use more recent/official vanilla F64 textures. These can be downloaded from the Faithful website, and should then be listed above Faithful 64x Modded and Faithful 32x Modded in your enabled resource pack list. It should be noted that, doing this may cause minor inconsistencies with modded textures based on older versions of vanilla textures.
